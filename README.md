# TC1-Rede-Neural-para-Previsao-de-Churn-com-Pipeline-Profissional-End-to-End
Pipeline end-to-end de previsão de churn para operadora de telecomunicações: MLP em PyTorch, baselines com Scikit-learn, rastreamento com MLflow, API com FastAPI e deploy em nuvem. Projeto acadêmico da FIAP · Tech Challenge Fase 1.

## Download do Dataset

Kaggle CLI
kaggle datasets download -d blastchar/telco-customer-churn -p data/raw/

Link: https://www.kaggle.com/datasets/blastchar/telco-customer-churn
